# .NET Cloud Architecture Labs

A collection of small labs and experiments exploring cloud architecture concepts using .NET.

## About

This repository contains various hands-on labs created while learning and experimenting with cloud services and distributed system patterns.

Each lab is placed in its own folder and focuses on a specific concept, integration, or architectural idea.

## Tech Stack

* .NET
* Cloud Services (e.g., storage, messaging, background processing)
* REST APIs

## Purpose

The goal of this repository is to practice cloud architecture patterns and understand how .NET applications integrate with cloud platforms.

## Notes

These labs are built for learning and experimentation and may evolve over time as new concepts and services are explored.
